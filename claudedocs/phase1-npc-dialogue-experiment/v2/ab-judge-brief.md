# A/B judge brief — which telling of each scene reads better?

You will be given FOUR scenes from a story. For each scene you get TWO versions of the
same scene — **Version 1** and **Version 2** — telling the same events with the same
people. They differ only in HOW the scene is told. Your job is to say, for each scene,
which version reads better AS STORY, and why.

Do not speculate about how either version was produced, or why there are two. Judge only
what is on the page.

## What "better" means here

For each scene, prefer the version that reads as **living, dramatized human talk** — people
speaking under pressure, evading, breaking, leaving things unsaid, present and acting on
the page — over a version that reads as **arranged or summarized information** — stakes
announced for the reader, truths declaimed as set-piece oration, or a cast accounted for
from outside in narrated summary rather than caught in the moment.

Read both versions of a scene fully before choosing. A version is not better for being
longer or more ornate — it is better for reading as a scene a reader lives through.

## For EACH of the four scenes

- Choose **Version 1** or **Version 2** as the better telling.
- Give ONE sentence of why, quoting or citing a specific line.
- If they are genuinely equal, you may say "tie" — but only if you truly cannot prefer one.

## Then, overall

- Across the four scenes, is **Version 1** or **Version 2** the better telling on the whole?
  (The two versions are consistent: whatever "Version 1" is in scene one, it is the same
  source throughout.) One sentence why.

## Output format

```
Scene 1 (<scene id>): better = Version 1 | Version 2 | tie — <one sentence, cite a line>
Scene 2 (<scene id>): better = Version 1 | Version 2 | tie — <one sentence, cite a line>
Scene 3 (<scene id>): better = Version 1 | Version 2 | tie — <one sentence, cite a line>
Scene 4 (<scene id>): better = Version 1 | Version 2 | tie — <one sentence, cite a line>
OVERALL better telling: Version 1 | Version 2 — <one sentence>
Note any version that read as if a scene had been spliced in from a different hand, if any: ...
```

That last line matters: if one version's scene felt stylistically out of step with good
storytelling (not just different), say so — but base your scene choices on which reads
better as lived story, not on which feels different.
