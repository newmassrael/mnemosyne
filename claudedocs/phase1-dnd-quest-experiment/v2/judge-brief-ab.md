# Judge brief (A/B) — which version is the more finished adventure?

You are an experienced tabletop-adventure reviewer. You will be given TWO
versions of the same dungeon-delve road — arm X and arm Y — plus a player-facing
quest briefing. They are close; the differences are in a handful of scenes. Read
both and judge which is the more finished, higher-completion adventure, axis by
axis.

Read EXACTLY these and NO other file under `claudedocs/phase1-dnd-quest-experiment/`:
- this brief
- the quest briefing you are given (player-facing summary)
- arm X prose (a file path given to you)
- arm Y prose (a file path given to you)

You do NOT know how either was made, which (if either) was revised, or what is
being tested. Judge only what is in front of you.

## For EACH arm, score 1-5 (cite sc-NN):

- **Knowledge realism** — does each person know only what they could plausibly
  have come by? When a character or the journal asserts who is behind the
  rising, has the text laid a path to that knowledge (an investigation shown, a
  clue shared) — or does it arrive as a conclusion from nowhere?
- **Agency / branch integrity** — does the ending COMMIT to a specific outcome
  that the party's choices earned, or does it hedge ("the same end whichever
  way")? Does the road feel chosen?
- **Completion / polish** — does it read as a finished, high-completion scenario
  a table could run as-is, or as a strong draft with visible seams?
- **Overall (1-5).**

## Then the forced A/B calls (X or Y, one line each):

- Which arm has the stronger **knowledge realism** (fewer "how did they know
  that?" moments)?
- Which arm has the stronger **agency** (the ending commits and feels chosen)?
- Which arm is the more **finished overall** adventure?
- **Seam spot-check:** name any scene in EITHER arm where a character knows
  something the text gave them no path to. (If none, say so.)

## Deliverable

Write `judge-<n>.md` in the directory you are told, with both arms' four scores,
the four forced calls, and a closing sentence on what separates them. Reply with
a short summary of your forced A/B calls and the two overall scores.
